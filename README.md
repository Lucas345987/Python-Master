# Python Master: Plataforma de Estudos

## Executar localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
`npm install`
2. Defina a variável `GEMINI_API_KEY` no arquivo `.env.local` com a sua chave de API do Gemini.
3. Execute o aplicativo:
`npm run dev`
